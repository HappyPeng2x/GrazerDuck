Shell commands

.help

List all shell dot-commands

.open :memory:

Reset to a fresh in-memory database

.timer on

Show query execution time

.mode table

Switch output to table format

Import (after clicking "Import File")

SELECT * FROM read_csv_auto('file.csv');

Auto-detect CSV types and headers

SELECT * FROM read_parquet('file.parquet');
SELECT * FROM read_json_auto('file.json');
CREATE TABLE t AS SELECT * FROM read_csv_auto('file.csv');

Export (via shell)

COPY (SELECT * FROM t WHERE id > 10) TO 'out.parquet';
COPY my_table TO 'out.json' (FORMAT JSON, ARRAY true);
